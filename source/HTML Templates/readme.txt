List of Template Parameters:

${swf}                Specifies url to swf file without extension.
${title}              Specifies title of HTML page.
${application}        Specifies name of application. Used as value for id attribute of HTML object tag.
${version_major}      Specifies major minimum required version of flash player.
${version_minor}      Now is always 0.
${version_revision}   Now is always 0.
${width}              Frame width.
${height}             Frame height.
${bgcolor}            Specifies value of background color in HTML hex format.
${expressInstallSwf}  Specifies SWF file which is used to install Flash Player.
